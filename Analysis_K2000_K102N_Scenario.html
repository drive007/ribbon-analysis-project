Proposed Consolidation Scenario Analysis

Replacements & Volumes

  • Use KURZ K2000 (117mm*) @ 245 THB
    • Replaces 8-394 (W/R 117mm): 800 rolls
    • Replaces 8-392 (W/R 85mm): 210 rolls
    • Total K2000 Rolls: 1,010
    • (*117mm assumed, validation needed for 85mm use)
  • Use KURZ K102N (110mm*) @ 175 THB
    • Replaces 8-393 (Wax 110mm): 550 rolls
    • Total K102N Rolls: 550
    • (*110mm price assumed available)

Projected Cost Calculation

K2000 Cost (1010 rolls * 245 THB): ~247,450 THB
K102N Cost (550 rolls * 175 THB): ~96,250 THB
Projected Total Annual Cost: ~343,700 THB
Current Total Annual Cost: ~828,500 THB
Estimated Annual Savings: ~484,800 THB (~58.5%)

Key Considerations for this Scenario

  • Reduces SKUs from 3 to 2.
  • Maintains correct ribbon type (Wax/Resin & Wax) for primary applications, minimizing durability risk.
  • Offers very significant savings (~58.5%).
  • Requires K2000 validation vs. SATO T102E.
  • Requires K102N validation vs. BEA YI B220N.
  • Requires validation/optimization for using 117mm K2000 on 85mm lines (waste/guides). (Medium Risk)
  • Requires confirmation of K102N 110mm price/availability.
  • Opportunity: Request K2000 in 85mm width from KURZ.

Action Plan Overview (For K2000 + K102N Scenario)

  • Confirm K102N 110mm Price & Availability with supplier.
  • Procure test samples: KURZ K2000 (117mm) and KURZ K102N (110mm, once price/availability confirmed).
  • Define a detailed test protocol for both K2000 (vs. Wax/Resin) and K102N (vs. Wax).
  • Execute comprehensive performance testing:
    • K2000 against current Wax/Resin ribbons (8-394, 8-392) on all relevant materials (JP PACK, Gowns, SPT, OEM), focusing on durability.
    • K102N against current Wax ribbon (8-393) on relevant materials (Wraps, Masks).
  • Assess the impact (waste, guide adjustments) of using 117mm K2000 on production lines currently using 85mm ribbons.
  • (Optional, based on waste assessment) Request a quotation for K2000 in 85mm width from KURZ.
  • Compile all test results, cost analysis (including potential waste from size mismatch), and overall scenario feasibility.
  • Hold a final decision meeting with management to determine Go/No-Go for this consolidation scenario.
  • If approved, develop and execute a detailed implementation plan, including printer recalibrations, stock transition, and staff training.

Analysis based on 2024 usage data and provided quotations/datasheets (Apr 2025 Prices).
Scenario assumes K102N 110mm price availability and uses K2000 117mm for W/R replacements. Dimension mismatch requires validation.
